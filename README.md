#Codiad Compass Plugin

Compass commands integration for [Codiad IDE](https://github.com/Codiad/Codiad)

##Requirements
- [Compass](http://compass-style.org/)

##Installation

- Download the zip file and unzip it to your plugin folder.

##Features

- Create
- Compile
- Clean
- Watch
- Log window
